Graphic_Engine
==============
Making my own graphic engine   
Project goal is to make a simple graphic engine
   
   
requirements
------------
OpenGL (3.3 or higher)   
GLFW 3 (3.3)   
GLM (0.9.9.5)   
   
   
Organization
------------
##### CoreEngine
- core_graphic    
##### ObjectEngine
- geometry
    * geometry
    * geometry_function
- component
- command
    * command_queue   
##### GraphicEngine
- window
    * openGL_window
    * window_setting
- shader
    * openGL_shader
- render
    * openGL_render      
##### PhysicsEngine
- hitbox
    * surface
    * hitbox
- transform
    * transform
    * compute_function
- collision
    * collision
